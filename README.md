# 🏥 Site Institucional - Clínica Fenix

Site institucional completo e profissional para a Clínica Fenix, especializada em saúde, medicina ocupacional e segurança do trabalho.

## 📋 Sobre o Projeto

Site desenvolvido com HTML5, CSS3/SASS e JavaScript, seguindo as melhores práticas de desenvolvimento web, com foco em:

- **Design Responsivo**: Adaptável a todos os dispositivos (desktop, tablet e mobile)
- **Performance Otimizada**: Carregamento rápido e eficiente
- **SEO Friendly**: Otimizado para mecanismos de busca
- **Acessibilidade**: Navegação facilitada e acessível
- **Animações Modernas**: Efeitos visuais suaves e profissionais

## 🎨 Identidade Visual

- **Cor Principal**: #215df9 (Azul)
- **Cor Secundária**: #FFFFFF (Branco)
- **Tipografia**: Inter (corpo) e Poppins (títulos)

## 📦 Como Compilar o SASS

```bash
# Compilar SCSS para CSS
sass scss/main.scss css/main.css

# Compilar e minificar
sass scss/main.scss css/main.min.css --style=compressed

# Watch mode (compilar automaticamente ao salvar)
sass --watch scss/main.scss:css/main.css
```
